# XTDB-chat

Chat with your (bitemporal) database.

## Prerequisites

Specify your OpenAI API key as an environment variable

```
export OPENAI_API_KEY=<YOUR_KEY_HERE>
```

## Usage

To run locally:

```
make install
make run
make test
```

To run in docker:

```
make docker
```

...and in another terminal...

```
make drun
```
